# 🚀 راهنمای Deploy روی Railway

## مرحله 1: ثبت‌نام
1. برو به [Railway.app](https://railway.app/)
2. روی **"Sign up"** کلیک کن
3. **"Sign up with GitHub"** را انتخاب کن
4. اجازه دسترسی به repository را بده

## مرحله 2: Deploy پروژه
1. روی **"New Project"** کلیک کن
2. **"Deploy from GitHub repo"** را انتخاب کن
3. Repository **"Hamidbarzin/Barzin_Crypto"** را انتخاب کن
4. روی **"Deploy Now"** کلیک کن

## مرحله 3: تنظیمات خودکار
Railway خودکار:
- ✅ Python 3.10 را تشخیص می‌دهد
- ✅ Dependencies را نصب می‌کند
- ✅ `render_main.py` را اجرا می‌کند
- ✅ دامنه رایگان می‌دهد

## مرحله 4: دریافت دامنه
1. بعد از deploy، دامنه شما آماده است
2. فرمت: `https://yourproject.railway.app`
3. روی دامنه کلیک کن تا وب‌سایت را ببینی
